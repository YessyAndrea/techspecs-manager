# TechSpecs Manager

Aplicación web sencilla para registrar y gestionar servidores con sus especificaciones técnicas.
Permite crear, listar y eliminar servidores usando una API REST simulada.

---

## Tecnologías usadas

- HTML5
- CSS3
- JavaScript (Fetch API)
- json-server / mockapi.io (API REST simulada)
- Git & GitHub

---

## Funcionalidades

- Registrar servidores con:
  - Nombre
  - CPU (mínimo 2 núcleos)
  - RAM (mínimo 4 GB)
  - Tipo de almacenamiento (SSD / HDD)
  - Presupuesto (máx. 700 €)
- Listar servidores registrados
- Eliminar servidores
- Validaciones en el formulario
- Persistencia de datos mediante API REST
